HTML と CSS を学び始めて気づいたこと

このブログを作りながら HTML と CSS を独学で勉強しています。最初は難しそうに思えましたが、実際に画面に反映される瞬間の達成感がクセになってきました。

勉強を始めたばかりのころ、HTML はただ「タグで囲むもの」という印象しかありませんでした。でも実際に書き始めると、タグひとつひとつに意味があって、それが構造を作り、CSS がそこに色や形を与えていくという流れがだんだん見えてきました。まるでスケルトンに肉付けをするような感覚です。

特に CSS の「box model」の概念を理解したときは、それまでの「なぜずれるの？」という疑問がすっきり解けた気がして、小さな感動がありました。margin と padding の違いを体で覚えると、レイアウトが急に思い通りになってくるんですよね。

まだまだ分からないことばかりですが、「動いた！」という瞬間の快感があるかぎり、続けていけそうです。次は JavaScript にも少しずつ触れてみようと思っています。
